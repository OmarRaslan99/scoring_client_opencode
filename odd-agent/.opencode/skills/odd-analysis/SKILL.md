## What I Do

J'aide l'agent a scorer une entreprise selon les 17 ODD avec une posture conservatrice.

## Rules

- Evaluer chaque ODD separement avec une echelle de -1 a 3.
- Utiliser 0 quand les sources ne montrent pas d'impact materiel clair.
- Ne jamais extrapoler sans preuve.
- Exiger une confidence de 0 a 100 pour chaque ODD.
- Ajouter une explanation courte uniquement pour les scores non nuls.
